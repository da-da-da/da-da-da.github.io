发表于 2020-09-05 | 分类于 二叉树

给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

示例 1:

[1,2,3],   [1,2,3]

输出: true
示例 2:

   1          1
  /            \
 2              2

[1,2],     [1,null,2]

输出: false

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/same-tree

  • 深度优先算法

    思路：对于两个节点，以他们为根节点的两个树，要判断这两个节点的左子树是否相等，右子树是否相等，对于每个节点都是如此判断，这是一个递归的过程，则使用深度优先算法，递归判断两个节点是否相等，其左右子树是否相等。

    复杂度：时间复杂度：O(min(m,n))，其中 m 和 n 分别是两个二叉树的节点数。对两个二叉树同时进行深度优先搜索，只有当两个二叉树中的对应节点都不为空时才会访问到该节点，因此被访问到的节点数不会超过较小的二叉树的节点数。

    空间复杂度：O(min(m,n))，其中 m 和 n 分别是两个二叉树的节点数。空间复杂度取决于递归调用的层数，递归调用的层数不会超过较小的二叉树的最大高度，最坏情况下，二叉树的高度等于节点数。

    判断什么时候要结束递归

    • 两个节点都为null，return true
    • 两个节点之一为null，return false
    • 节点不相等，return false

  • 广度优先算法

    这个方法是看了题解才知道可以用广度

    思路：从根节点开始进行层序遍历，先看出队列的值是否相同，再看每个出队列节点的的左右子节点的结构，如果不同则false，如果相同入栈。直到一个队列为空

    复杂度分析

    时间复杂度：O(min(m,n))，其中 m 和 n 分别是两个二叉树的节点数。对两个二叉树同时进行广度优先搜索，最坏是遍历完节点数较小的那个

    空间复杂度：O(min(m,n))，其中 m 和 n 分别是两个二叉树的节点数。空间复杂度取决于队列中的元素个数，队列中的元素个数不会超过较小的二叉树的节点数。
